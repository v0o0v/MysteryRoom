# 🧩 Project Name: Mystery Room (가제)
> **Genre:** 3D 방탈출 / 서스펜스 / 퍼즐 / 추리

---

## 📌 Overview
**"가장 익숙한 공간이 가장 낯선 곳이 된다."** 본 프로젝트는 현대적인 한국의 원룸을 배경으로 한 1인칭 미스터리 탈출 게임입니다. 플레이어는 제한된 공간 속에서 사물과 상호작용하며 숨겨진 단서를 조합하고, 논리 및 물리 기반의 퍼즐을 해결하여 잠긴 공간을 탈출해야 합니다.

### Key Features
* **Physics Interaction:** `Raycast` 기반의 상호작용 시스템을 통해 서랍 열기, 물건 집기 등 정교한 물리 조작 구현.
* **Immersive Atmosphere:** Unity `URP(Universal Render Pipeline)`를 활용한 고품질 라이팅과 포스트 프로세싱으로 서스펜스 분위기 조성.

## 🛠 Tech Stack
- **Engine:** Unity 6000.3.10f1 LTS (Universal Render Pipeline)
- **Render Pipeline:** Universal Render Pipeline (URP)
- **Language:** C#
- **Target Platform:** PC (Windows)

## 💻 Installation
1. Clone the repository.
2. Open with Unity Hub (Version 6000.3.10f1 Recommended).
3. Play `Scenes/MainScene.unity`.
